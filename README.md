# WikiWebMap (WikiWeb Explorer)

Explore Wikipedia topics as an interactive graph: expand nodes, search for connections between topics, and visualize link context.

Live: `https://WikiWebMap.flyingchangesfarm.net`

## Quickstart

### Prerequisites
- Node.js 20+

### Install
- `npm ci`

### Run
- `npm run dev`

### Build
- `npm run build`

## Controls (high level)
- Drag canvas to pan; scroll to zoom.
- Click a node to open details and actions.
- Shift+Click selects up to 2 nodes to search for connections (pathfinding).
- Alt/Option+Drag box-selects nodes for bulk actions.
